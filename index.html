Cerberus

Code

Java program for cracking MD5 and SHA hashes. Types of cracking are wordlist, bruteforce, or rainbow-table. Contains aditional tools for hashing cleartext passwords, distinguishing hash types, or creating rainbow-tables. Your work can be saved into projects, so next time when you open the application you can continue where you left off.

Duck to Spark

Web, Code

USB Rubber Ducky is well known device with big community and lot of scripts, but with high price $49.99 it isn't accessible to everyone. On the other hand DigiSpark with price around $1 is cheap alternative but because it's based on Arduino, it makes Ducky Script useless there. CCNA netacad toolkit

Code

This toolkit includes scraper, for scraping questions and correct answers from web, and then two modules:

- One will generate JavaScript which you can paste into exam window, and correct answers will change your cursor to 'loading' when you hover over them,

- Second module will open new Firefox window where you will be asked to login into netacad and start exam, on first question type 'start' into console and script will complete exam for you.
